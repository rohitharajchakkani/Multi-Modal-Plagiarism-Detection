About PlagiarismAI

A state-of-the-art Multi-Modal Plagiarism Detection System engineered for comprehensive analysis.

📖 Project Overview

PlagiarismAI is an advanced multi-modal auditing platform designed to detect similarities across different file formats. Unlike traditional detectors that only look at raw text copying, PlagiarismAI combines high-performance natural language processing (NLP), semantic machine learning, and advanced computer vision layout analysis to scan document text structure, styles, colors, layouts, and live web resources.

🔬 Core Comparison Modes

📄

1. Text Document Comparison

Analyzes digital text content using a hybrid model:

  • TF-IDF Vectors (40%): Token occurrences frequency weighting.
  • Cosine Similarity (30%): Word counts direction similarity.
  • SBERT Transformers (30%): Underline semantic paraphrase understanding.

🖼️

2. Visual Document Comparison

Audits visual formatting, design structures, and posters using:

  • OCR Text Analysis (40%): Tesseract image text alignment.
  • Layout Contour Correlation (30%): Canny boundary shapes.
  • dHash Image Hashing (20%): Structural scale-invariant hashing.
  • HSV Color Histograms (10%): Color profiles matching.

🌐

3. AI Web Source Detection

Scans uploaded text or visual documents against live web indices:

  • API Web Search Crawlers: Queries via Tavily API / SerpAPI.
  • Title & Metadata Heuristics: Auto-extracts titles, keywords, headings.
  • Reliability Indexes: Auto-categorizes journals (arXiv, ResearchGate) vs. blogs.

⚙️ How It Works

  1. Select a comparison mode based on document type (Text, Visual, Web-backed).
  2. Upload digital files or paste raw inputs. For images/PDFs, OpenCV filters denoise and align images prior to PyMuPDF and Tesseract OCR text extraction.
  3. The similarity pipelines execute weightings (NLP vectorizers, SBERT embedding vectors, or HSV/dHash structural layout metrics).
  4. Calculated results are compiled and indexed into database records.
  5. A comprehensive similarity breakdown report is generated, logging unread notifications for safety metrics.
